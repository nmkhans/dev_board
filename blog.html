Question-1: What are the different ways to select an element in dom?

There are different ways to select element in dom. Example: tag selector, class selector, id selector, css query selector ect. Among them class, id and query selectors are commonly used most ofter

Question-2: What are the different between innerHTML, innerText and textContent?

innerHTML gives us html element, text, comments everything, innerText gives us only text of and element and its child element, textContent gives us text along with comments and stuff like that.

Question-3: What is event delegation in dom?

Event delegation is a technique that allow us to apply event listener to a parent element and listen to action that is happeing in the child element without applying event listener to all child element indivisually.

Question-4: What is event bubble in dom?

Event bubble is besically when we apply a event listener to an element, when the event is triggered the event bubbles up to all its parent element. So an event happening in child element can be listen in the parent elements cause the event bubble up. Thats event bubble.

Question-5: How to create add and remove element using js?

we can create an element using document.create method, add it to the dom using element.append or element.appendChild method and delete from dom using element.removeChild method.
